# Intent

## Prompt

Build a stateless HTTP API that performs scientific math operations — arithmetic, trigonometry, logarithms, powers, statistics, and unit conversions. Any HTTP client can consume it without installing a math library. The calculator prioritises correctness, precision, and clear error reporting over raw throughput.

## Summary

A stateless Scientific Calculator REST API built with Python 3.13 and FastAPI, managed with uv. It exposes endpoints for arithmetic, powers/roots, trigonometry, logarithms, statistics, constants, and unit conversions — all returning structured JSON responses with consistent error handling. The project includes comprehensive pytest-based testing with 90%+ coverage.
