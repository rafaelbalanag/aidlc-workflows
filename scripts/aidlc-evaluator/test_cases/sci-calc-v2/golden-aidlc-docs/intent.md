# Intent

## Prompt

Build a Scientific Calculator API — a stateless HTTP API that performs scientific math operations (arithmetic, trigonometry, logarithms, powers, statistics, and unit conversions) using Python 3.13, FastAPI, and uv package manager. The full vision is in vision.md and the technical environment is in tech-env.md.

## Summary

Deliver a stateless HTTP REST API that exposes scientific math operations — arithmetic, powers/roots, trigonometry, logarithms, statistics, constants, and unit conversions — via JSON endpoints. The service is built with Python 3.13, FastAPI, and the uv package manager, targeting correctness, precision, and clear error reporting. It serves as a complete greenfield MVP with comprehensive test coverage (≥90% lines) and structured error envelopes for all failure cases.
